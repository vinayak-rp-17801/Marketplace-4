# FINMA Circular 2023/1 Compliance for Log360 Cloud

**Tagline:** Fulfill FINMA's operational risks and resilience requirements with centralized log monitoring and analytics from Log360 Cloud.

## Overview
The FINMA Circular 2023/1 — Operational Risks & Resilience Compliance extension for Log360 Cloud helps Swiss financial institutions meet FINMA's stringent requirements for IT risk management, operational resilience, and incident reporting. Log360 Cloud collects and correlates logs from banking applications, cloud environments, network devices, and identity systems, providing continuous visibility into security events. Security and compliance teams can monitor critical control points, detect operational anomalies, and generate detailed audit evidence to satisfy FINMA's supervisory expectations and internal governance obligations.

## Key Features
- Generate comprehensive audit reports covering IT operational risk controls, user access monitoring, and system availability events aligned to FINMA Circular 2023/1.
- Track privileged user activity, administrative changes, and access control modifications across financial IT systems in real time.
- Leverage AI-powered threat insights to correlate security events and identify operational risk indicators before they escalate into reportable incidents.
- Export audit evidence in PDF and CSV formats to support FINMA submissions, internal risk committees, and external auditor reviews.
