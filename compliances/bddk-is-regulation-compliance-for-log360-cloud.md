# BDDK IS Regulation Compliance for Log360 Cloud

**Tagline:** Meet Turkey's banking information systems security requirements with unified log monitoring and reporting from Log360 Cloud.

## Overview
The BDDK IS Regulation (Banks' Information Systems) Compliance extension for Log360 Cloud helps Turkish banks and financial institutions regulated by the Banking Regulation and Supervision Agency (BDDK) meet the information systems security, access control, and audit requirements of the BDDK IS Regulation. Log360 Cloud continuously collects and correlates logs from core banking systems, payment platforms, cloud environments, and network infrastructure, providing security teams with detailed visibility into IT security events. Compliance teams can monitor access to sensitive financial data, detect anomalies, and generate reports for BDDK examinations.

## Key Features
- Generate BDDK IS Regulation-aligned audit reports covering access management, system integrity monitoring, and security incident detection in banking environments.
- Track privileged user activities, administrative system changes, and access to core banking applications in real time.
- Use ML-powered anomaly detection to identify unusual patterns in banking system access and flag potential insider threats or compromised accounts.
- Export audit evidence in PDF and CSV formats for BDDK on-site inspections and internal IT audit reviews.
