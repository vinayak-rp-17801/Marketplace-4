# Italian Cybersecurity Perimeter (DPCM 81/2021 + Law 133/2019) Compliance for Log360 Cloud

**Tagline:** Protect national security assets and meet Italy's Cybersecurity Perimeter obligations with continuous monitoring from Log360 Cloud.

## Overview
The Italian Cybersecurity Perimeter (DPCM 81/2021 + Law 133/2019) Compliance extension for Log360 Cloud supports Italian public bodies, operators of critical national networks, and strategic enterprises in fulfilling the security monitoring and incident reporting requirements of Italy's National Cybersecurity Perimeter. Log360 Cloud ingests and correlates logs from critical systems, network infrastructure, cloud environments, and identity platforms, providing real-time visibility into events affecting perimeter assets. Security teams can rapidly detect incidents, perform forensic investigations, and produce the documentation required by ACN and CVCN oversight.

## Key Features
- Generate compliance reports covering security monitoring events for perimeter-designated assets, aligned to DPCM 81/2021 requirements.
- Track access to critical systems, detect unauthorized changes, and identify anomalous user behavior across perimeter infrastructure in real time.
- Leverage preconfigured correlation rules to detect advanced threats and supply chain attacks targeting national cybersecurity perimeter assets.
- Export audit evidence in PDF and CSV formats to support ACN notifications and CVCN security assessments.
