# BAIT Compliance for Log360 Cloud

**Tagline:** Meet BaFin's IT supervisory requirements for banks with comprehensive log monitoring and audit reporting from Log360 Cloud.

## Overview
The BAIT (BaFin Supervisory Requirements for IT in Banks) Compliance extension for Log360 Cloud helps German credit institutions meet BaFin's requirements for IT governance, access management, IT operations security, and incident management as defined in BAIT. Log360 Cloud collects and correlates log data from banking applications, core systems, cloud platforms, and network infrastructure, providing continuous monitoring of IT security events. Compliance and IT security teams can track user access, detect anomalies, and generate the audit-ready evidence required for BaFin inspections and internal IT audits.

## Key Features
- Generate predefined BAIT compliance reports covering IT operations security, identity and access management, and security incident monitoring.
- Track privileged user activities, system configuration changes, and access to critical banking applications in real time.
- Use ML-powered UEBA to identify anomalous behavior by bank employees and system accounts that may indicate insider threats or compromised credentials.
- Export audit evidence in PDF and CSV formats for BaFin supervisory examinations and internal IT risk committee reviews.
