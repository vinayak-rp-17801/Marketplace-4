# Banco de España Circular 2/2016 Compliance for Log360 Cloud

**Tagline:** Meet Banco de España's IT security supervisory requirements with centralized monitoring and audit reporting from Log360 Cloud.

## Overview
The Banco de España Circular 2/2016 (as amended by 3/2022) Compliance extension for Log360 Cloud helps Spanish credit institutions meet the IT governance, access management, and operational security requirements set out in Banco de España's supervisory circular. Log360 Cloud collects and correlates logs from banking applications, core systems, cloud platforms, and network infrastructure, providing security and compliance teams with continuous visibility into IT security events. Teams can detect anomalies, investigate incidents, and generate the audit evidence required for Banco de España inspections and EBA supervisory reporting.

## Key Features
- Generate predefined audit reports aligned to Banco de España Circular 2/2016 requirements covering access management, IT operations security, and incident monitoring.
- Track privileged user activities, core banking system changes, and access events across Spanish banking IT environments in real time.
- Use ML-powered UEBA to detect unusual access patterns and flag potential insider threats in banking system environments.
- Export compliance documentation in PDF and CSV formats for Banco de España supervisory examinations and internal IT risk governance.
