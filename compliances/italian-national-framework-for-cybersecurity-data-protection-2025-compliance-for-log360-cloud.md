# Italian National Framework for Cybersecurity & Data Protection 2025 Compliance for Log360 Cloud

**Tagline:** Align with Italy's 2025 cybersecurity and data protection best practices using integrated monitoring and analytics from Log360 Cloud.

## Overview
The Italian National Framework for Cybersecurity & Data Protection 2025 Compliance extension for Log360 Cloud enables Italian organizations to align with ACN's updated national cybersecurity framework across risk management, data protection, and incident response domains. Log360 Cloud provides a unified view of security events across on-premises, cloud, and hybrid environments, enabling continuous monitoring and rapid incident response. Teams can map controls to framework categories, identify gaps, and maintain the evidence required for self-assessment and regulatory reviews.

## Key Features
- Generate compliance reports mapped to Italian National Framework categories covering identify, protect, detect, respond, and recover functions.
- Monitor user access events, data handling activities, and system changes across all IT environments in real time.
- Leverage AI-powered threat insights with MITRE ATT&CK mapping to contextualize detected threats and guide incident response actions.
- Export audit-ready evidence in PDF and CSV formats for internal risk assessments and external regulatory submissions.
