# Swiss Telecommunications Act (TCA) Compliance for Log360 Cloud

**Tagline:** Ensure telecommunications security and data retention compliance with continuous log monitoring from Log360 Cloud.

## Overview
The Swiss Telecommunications Act (TCA) Compliance extension for Log360 Cloud helps Swiss telecom providers and digital service operators meet the security monitoring, data protection, and lawful interception-related obligations under the TCA. Log360 Cloud ingests logs from network infrastructure, access systems, cloud platforms, and communication services, enabling organizations to maintain detailed audit trails of system events and user activities. Security teams can detect unauthorized access, monitor network integrity, and generate regulatory-grade reports to demonstrate compliance with TCA requirements.

## Key Features
- Generate predefined audit reports covering network access events, system changes, and data handling activities relevant to TCA obligations.
- Monitor authentication logs, network device events, and user activity across telecommunications infrastructure in real time.
- Configure threshold-based alerts for TCA-relevant security events such as unauthorized system access, configuration changes, and data exfiltration attempts.
- Export compliance reports in PDF and CSV formats suitable for regulatory submissions to BAKOM and internal governance reviews.
