# PGSSI-S Compliance for Log360 Cloud

**Tagline:** Implement France's General Health Information System Security Policy with unified monitoring and reporting from Log360 Cloud.

## Overview
The PGSSI-S (Politique Générale de Sécurité des Systèmes d'Information de Santé) Compliance extension for Log360 Cloud enables French healthcare organizations, hospitals, and health information system operators to implement the security monitoring and access control requirements of the PGSSI-S framework published by ANS. Log360 Cloud collects and correlates logs from health information systems, electronic health records, cloud platforms, and network infrastructure, providing security teams with continuous visibility into patient data access and system events. Compliance teams can track PGSSI-S control adherence and generate audit evidence for HAS and ANS oversight.

## Key Features
- Generate predefined compliance reports aligned to PGSSI-S security requirements covering patient data access, user authentication, and clinical system monitoring.
- Monitor health information systems for unauthorized access to patient records, anomalous user behavior, and security policy violations in real time.
- Use ML-powered UEBA to identify insider threats and compromised clinical accounts accessing sensitive health data.
- Export PGSSI-S compliance documentation in PDF and CSV formats for ANS reporting and internal healthcare information security governance.
