# NCSC Minimum ICT Standard (IKT-Minimalstandard) Compliance for Log360 Cloud

**Tagline:** Meet Switzerland's NCSC minimum ICT security requirements with continuous monitoring and audit reporting from Log360 Cloud.

## Overview
The NCSC Minimum ICT Standard (IKT-Minimalstandard) Compliance extension for Log360 Cloud helps Swiss federal agencies and operators of critical infrastructure implement the NCSC's baseline cybersecurity measures for IT systems. Log360 Cloud collects and correlates logs from servers, endpoints, network devices, and cloud environments, enabling security teams to continuously monitor controls across the five IKT-Minimalstandard function categories: identify, protect, detect, respond, and recover. Compliance teams can assess their control coverage, remediate gaps, and generate audit evidence for NCSC reviews.

## Key Features
- Generate compliance reports mapped to IKT-Minimalstandard function categories covering asset monitoring, threat detection, access control, and incident response.
- Detect threats in real time using correlation rules and ML-powered anomaly detection across Swiss federal and critical infrastructure IT environments.
- Monitor authentication events, system configuration changes, and privileged access activities to verify protective control effectiveness.
- Export audit-ready reports in PDF and CSV formats for NCSC reporting and internal cybersecurity governance assessments.
