# TIBER-IT National Guide (Updated 2025) Compliance for Log360 Cloud

**Tagline:** Support Italy's threat-led red teaming exercises and strengthen cyber resilience with detection and monitoring from Log360 Cloud.

## Overview
The TIBER-IT National Guide (Updated 2025) Compliance extension for Log360 Cloud helps Italian financial institutions and critical infrastructure operators prepare for and respond to TIBER-IT exercises coordinated by Banca d'Italia. Log360 Cloud provides the detection and monitoring capabilities required to assess blue team performance during adversarial simulation exercises, correlating logs from IT systems, cloud environments, and network infrastructure to identify attack paths and detection gaps. Security teams can use Log360 Cloud to verify control coverage and generate documentation for TIBER-IT supervisory reporting.

## Key Features
- Generate reports documenting security event detection, alert timelines, and blue team response activities during TIBER-IT exercises.
- Monitor attacker simulation paths across Active Directory, endpoints, network devices, and cloud environments in real time.
- Evaluate MITRE ATT&CK detection coverage against TIBER-IT threat intelligence scenarios to identify gaps in the security monitoring program.
- Export TIBER-IT exercise reports in PDF and CSV formats for Banca d'Italia after-action reviews and internal security resilience governance.
