# CNMV / TIBER-ES Compliance for Log360 Cloud

**Tagline:** Strengthen Spanish financial market cyber resilience and support TIBER-ES exercises with detection capabilities from Log360 Cloud.

## Overview
The CNMV / TIBER-ES (Threat Intelligence-Based Ethical Red Teaming — Spain) Compliance extension for Log360 Cloud supports Spanish financial market participants and critical service providers in preparing for and measuring the effectiveness of TIBER-ES threat-led penetration testing exercises coordinated by the Banco de España and CNMV. Log360 Cloud provides the security monitoring and detection capabilities needed to assess blue team performance during adversarial simulations, correlating log data to surface attack paths and detection gaps. Security teams can evaluate detection coverage and document blue team responses for supervisory review.

## Key Features
- Generate reports documenting security event detection, alert timelines, and blue team response activities during TIBER-ES exercises.
- Monitor simulated attacker activity across Active Directory, endpoints, cloud environments, and network infrastructure in real time.
- Evaluate MITRE ATT&CK detection coverage against TIBER-ES threat intelligence scenarios to identify monitoring gaps.
- Export TIBER-ES exercise documentation in PDF and CSV formats for Banco de España and CNMV after-action reviews.
